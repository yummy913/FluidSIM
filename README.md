
install the repo with <pre>```git clone https://github.com/yummy913/FluidSIM.git```</pre>

then enter the wp directory <pre>```cd FluidSIM/wp/```</pre>

and start the fluid simulation <pre>```cargo run```</pre> to start the fluid simulation

functionality:

you are able to move both emitters by clicking and dragging on them and you can change the settings using the controls on the top left
